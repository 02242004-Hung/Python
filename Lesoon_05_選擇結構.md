## 判斷該年是否為閏年
```
條件1：逢4閏而且逢100不閏
條件2：逢400閏
```
```
a = int(input("請輸入年："))
if(a%4 == 0):
  if(a%100 == 0):
    if(a%400 == 0):
      print("此年是閏年")
    else:
      print("此年不是閏年")
  else:
    print("此年是閏年")
else:
  print("此年非閏年")
```
